# Privacy Policy

Except for the specific cases listed below, this program communicates only over the local network for the purpose of
sending/receiving sACN data.

## Autoupdater

If the autoupdater is enabled, this program will, on launch, request information about the latest release of the
program from GitHub. No information is sent to GitHub not required by a standard HTTP request.

## Error Reporting
If this program encounters an error or crashes, information about the problem is sent to Sentry.io. See their 
privacy policy [here](https://sentry.io/privacy/). Information collected includes:
- Your operating system's name, version, and build
- This program's version and build
- This program's memory contents
