# Doc notes

3rd_party.md is auto generated by `3rd_party.py` which scans vcpkg and npm packages for SPDX license files. SPDX files in
`licenses/` are also included.

Screenshots are taken on Windows with ShareX. Web UI screenshots are taken with Chrome using Developer Tools with the
device dimensions set to "iPhone SE" (375x667). Include both light and dark mode screenshots.

The doc theme includes Bootstrap, so the same Bootstrap classes used in the Web UI are available in doc. When referring
to a UI element, use the Bootstrap class to style the element:

```markdown
Press [Start]{.btn .btn-success} to begin transmitting.
```

Tabs are styled as a button in doc even though they are navs in the Web UI. Icons are also available:

```markdown
Press [<i class="bi bi-sliders"></i> Faders]{.btn .btn-tab} to show faders mode.
```
