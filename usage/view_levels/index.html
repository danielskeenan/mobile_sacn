View Levels

View Levels mode shows all sACN levels on the network and their merge result.

  • Usage

  • Modes

    • List

    • Grid

Usage

If any sACN sources are active on the network, their levels are shown for the given universe. If more than one sACN source is active, the levels from all sources is merged following the standard sACN merge rules. Each source is assigned a different color; the color legend is shown by opening the “Sources” dropdown.

Note

Per-address-priority is not currently supported in the View Levels screen.

Modes

Multiple display modes are available.

List

Press to show list mode.

The given universe is shown as a list of bars, where each address has a bar showing its current level. The color of the bar corresponds to the color of the winning source.

Grid

Press to show grid mode.

The given universe is shown as a grid with the first address in each row shown on the left. The number of columns in each row is configurable. Much like List mode, the color of the bar corresponds to the color of the winning source.
